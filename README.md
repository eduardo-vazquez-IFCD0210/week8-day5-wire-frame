### EJERCICIO WIREFRAMING (MAQUETACIÓN LAYOUT)
1- Usando la aplicación online wireframe.cc (u otra que tú conozcas, como Figma) genera la maquetación [bajafidelidad] para dos páginas de una tienda online:
- Página de inicio
- Página de producto
Añade notas de información donde lo creas necesario, para ti o tu equipo (atajo tecla A).

2- Versión que no hayas creado (móvil o escritorio): en wireframe.cc hay que crear una nueva página y copiar y pegar los elementos. En Figma no hace falta.

+EXTRA:3- A quien le dé tiempo, concretar con html-css esa página web (resultado final) [prototipo de alta fidelidad]

### EJERCICIO CLASIFICACION DE CATEGORIAS.
Tenemos una tienda de antigüedades y nos llega un camión con los siguientes elementos:
silla barroca, sellos especial olimpiadas, anillo s.XVII, pelicula resumen 16mm Star Wars, tren de hojalata, proyector de cine super-8 mm, reproducción numerada póster Alfons Mucha, póster película Pulp Fiction, pendientes modernistas 1910, monedas francesas s.XIX, radio de válvulas, cruz oro del barroco, camión de madera, biombo de madera, discos de vinilo de jazz, cromos Disney años 60, escritorio Luis XV, cámara cine Bolex H16 años 60, cuadro payaso óleo, juguete radio miniatura
Nos piden que coloquemos cada elemento en su categoría correspondiente. ¿Cuántas categorías crearías y dónde ubicarías cada objeto? (se entiende que cuantas menos mejor, y mejor que no haya una categoría con un solo elemento)
